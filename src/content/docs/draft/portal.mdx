title: 포탈
description: React 포탈을 사용하여 컴포넌트를 원하는 위치에서 렌더링하는 방법을 알아봅니다.
---

모달 다이얼로그는 document 루트에서 렌더링되어야 함
툴팁은 body의 끝에서 렌더링되어야 함

물론 useEffect로 DOM 노드를 추가하고 제거할 수 있겠지
하지만 createPortal을 사용하면 더 편리하게 할 수 있음. (일반적인 패턴이므로 이런걸 제공하는거지)

사용법은 간단함. JSX를 렌더링할때 createPortal로 감싸면 됨. 그리고 두 번째 인자로 어디에 렌더링할지 정함
즉, Modal 컴포넌트는 createPortal(JSX, document.body) 이걸 반환하면 되지

모달이나 툴팁을 사용하는 라이브러리는 내부적으로 createPortal을 사용할거임.
그러므로 어느 시점에서 이 API를 사용해야 할 수도 있음.

일반 자식 컴포넌트처럼 부모 컴포넌트에 의해 재렌더링된다고 함. 이건 실험한번 해보자. 그냥 렌더링되는 위치만 다를뿐 자식 컴포넌트임은 변함없기 때문.
