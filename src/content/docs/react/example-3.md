## 마크다운 테스트 콘텐츠

### 개요

이 문서는 마크다운 테스트를 위한 샘플 콘텐츠입니다.

### 엘릭서 코드 예시

다음은 엘릭서로 작성된 간단한 함수 예시입니다:

```elixir
defmodule HelloWorld do
  def greet(name) do
    IO.puts("안녕하세요, #{name}님!")
  end
end

HelloWorld.greet("홍길동")
```

#### 특징

- 간결한 문법
- 함수형 프로그래밍 지원
- 높은 확장성

### 결론

마크다운과 엘릭서를 활용하여 문서 작성과 코드 예시를 효과적으로 표현할 수 있습니다.
